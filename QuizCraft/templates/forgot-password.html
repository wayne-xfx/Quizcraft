========================================================= BACKEND NOTE: Forgot Password API Endpoint 1. On form submit, send a POST request to the backend API (e.g., /api/forgot-password) with the 'emailInput' value. 2. The backend queries the database to see if the email exists. 3. If found: Generate a secure, time-limited reset token, save it to the database, and trigger the email service. Respond with a success status so the frontend routes the user to /check-email. 4. If NOT found: Return a specific status code so the frontend knows to route the user to /email-not-found. =========================================================
