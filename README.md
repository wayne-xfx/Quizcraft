# Quizcraft - A Web-Based Quiz Application 


QuizCraft is a web-based quiz platform designed to help users create, manage, and share quizzes easily and efficiently.Built with usability and privacy in mind, it allows users to control access through private and shareable quizzes.

##  Features

- Community quizzes focused in IT trivias
- Create custom quizzes with multiple-choice & identification questions
- Share quizzes via unique links
- Smart Review Feature

##  Tech Stack

- **Frontend:** CSS
- **Backend:** Node.js
- **Database:** MongoDB
- **Authentication:** JWT Auth
